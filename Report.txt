¡IMPORTANTE!
Pasos previos para visualizar el proyecto:
	- Importar la base de datos (main_db.sql)
	- Crear el usuario (admin) con la contraseña (P@ssw0rd) y añadirle todos los privilegios dentro de nuestra base de datos (main_db)
	*EXTRA - SERVICIO PHP*
	- A través de una terminal, dirigirnos a la carpeta donde se encuentran los ficheros y iniciar el servicio local de PHP tal que así:
		php -S localhost:8000
